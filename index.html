SC-900: Enable Multi-Factor Authentication (MFA) in Azure AD

Problem Statement: Configure MFA for a user account in Azure AD to enhance security for cloud resource access.

Approach: Accessed Azure AD via the Azure Portal, enabled MFA for a test user, and configured authentication methods using the Microsoft Authenticator app. Tested MFA by logging in and verifying the authentication prompt.

Tools Used: Azure Portal, Microsoft Authenticator App, Azure AD

Key Lessons Learned: Understood how MFA strengthens security by adding an extra authentication layer and its role in compliance.
